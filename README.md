# Testory

**Testory** — desktop-приложение для управления тест-кейсами и циклами тестирования, ориентированное на сохранение истории выполнения и воспроизводимость результатов.

Проект находится на ранней стадии разработки и закладывает архитектурную основу для:
- работы с тест-кейсами как шаблонами,
- выполнения тестов в рамках циклов,
- хранения полной истории прогонов,
- последующего анализа и отчётности.

---

## ? Ключевые принципы

- **История первична**  
  Факты выполнения тестов не переписываются задним числом.

- **Разделение шаблонов и фактов**  
  TestCase — описание.  
  Execution — факт выполнения.

- **Десктоп-ориентированный workflow**  
  Приложение создаётся как standalone-инструмент, без зависимости от веб-инфраструктуры.

- **Простая и контролируемая архитектура**  
  Без магических состояний, с явной навигацией и предсказуемым UI.

## ? Запуск

Требования:
- установленный JDK 23+
- установленный Maven

Сборка и запуск:
```bash
mvn clean javafx:run
```
